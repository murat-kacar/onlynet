# Tutorials

Tutorials are for learning by doing.

Use this folder for guided onboarding flows that help a reader understand
TabFlow progressively rather than solve a narrow production task.

Current tutorials:

- [`./getting-started.md`](./getting-started.md) — suggested reading
  order for someone new to the TabFlow design.
- [`./local-development.md`](./local-development.md) — how a
  contributor stands up the platform, platform-worker, and tenant
  hosts on their own machine for development and tests.
