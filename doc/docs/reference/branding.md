# Branding

## Brand Stack

- **Company:** `CafeTech`
- **Product:** `TabFlow`
- **Category:** `Multi-tenant Admin Panel`
- **Platform UI:** `Management Console`

## Core Direction

- calm
- operational
- trustworthy
- clear

## One-Liner

`CafeTech builds the operating system for modern table service.`

## Positioning

`TabFlow` is a multi-tenant admin panel for running restaurant operations across customer environments.

## Product Line

| Surface | Name |
| --- | --- |
| platform admin | `TabFlow Management Console` |
| tenant admin/staff | `TabFlow Console` |
| guest ordering | `TabFlow Menu` |

## Voice

- short sentences
- low-hype language
- confident, not flashy
- practical over promotional

## Visual Direction

- light surfaces
- deep teal primary
- dense but orderly layouts
- no purple-gradient SaaS look
- scan-first, read-second
- state shown through hierarchy, color, and grouping

## UX Rules

- short labels, not paragraphs
- one dominant action per screen
- tables for control, cards for summary
- operators should scan, not study
- empty states stay useful and calm

## Reference Models

- `Shopify Plus Organization Admin` for multi-store / multi-tenant clarity
- `Stripe Dashboard` for calm operational density
- `Cloudflare` for control-plane seriousness
- `Okta Admin Console` for administrative trust and discipline
