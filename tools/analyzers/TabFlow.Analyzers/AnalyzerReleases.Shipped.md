; Shipped analyzer releases.
;
; This file is a normative artefact required by the Roslyn analyzer
; tooling rule RS2008. It lists every diagnostic ID that has shipped
; in a tagged release of TabFlow, grouped by version. The format is
; the convention defined by the Roslyn SDK; see the rule documentation
; at:
;   https://github.com/dotnet/roslyn-analyzers/blob/main/src/Microsoft.CodeAnalysis.Analyzers/ReleaseTrackingAnalyzers.Help.md
;
; The file remains empty until the first tagged release of TabFlow
; (AD-0011 SemVer policy). Diagnostic IDs that ship in the first
; release are moved here from AnalyzerReleases.Unshipped.md as part
; of the release PR.
